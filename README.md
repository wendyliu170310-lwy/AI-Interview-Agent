# AI-Interview-Agent
AI-powered interview evaluation and question generation workflow based on job descriptions and candidate backgrounds. Designed for fast-paced, execution-driven hiring teams.

---

# 面试助手 Skill

> 基于岗位 JD + 候选人背景，自动生成结构化面试评估与深度追问问题。

专为 **中型公司快节奏、重执行** 的招聘场景设计。

通过标准化面试流程，帮助面试官：

* 提升招聘准确率
* 降低主观偏差
* 快速识别候选人真实能力
* 自动生成深度追问
* 发现简历疑点与风险项

---

# Features

## ✅ 自动解析岗位 JD

自动识别：

* 必备能力
* 加分项
* 行业背景
* 软技能要求
* 数字化指标（预算 / 团队规模 / GMV 等）

---

## ✅ 候选人适配度评估

自动输出：

* 候选人画像
* 星级匹配度
* 优势分析
* 风险分析
* 稳定性判断

---

## ✅ 智能生成面试问题

每个问题包含：

* 主问题
* 深度追问方向
* 判断标准
* 加分信号
* 减分信号

支持：

* STAR 面试法
* 执行力验证
* 问题拆解能力
* 行业适配度验证
* 动机与稳定性验证

---

## ✅ 自动识别简历疑点

自动发现：

* 数字夸大
* 职责模糊
* 时间线缺口
* 数据存疑
* 职级矛盾
* 行业漂移

并生成：

```text
🔴 疑点描述
简历原文
建议追问
```

---

## ✅ 公司背景辅助验证

自动补充候选人过往公司：

* 行业定位
* 公司规模
* 核心产品
* 岗位匹配合理性

用于辅助真实性验证。

---

# 输出结构

Skill 固定输出：

```text
## 候选人画像

## 适配度评估

## 风险

## 定制面试问题

```

# 面试设计原则

## STAR 法则

```text
Situation → Task → Action → Result
```

---

## 深度追问机制

不是只问一句。

每个问题必须：

* 可继续追问
* 能验证真实性
* 能拆解执行过程

---

## 强调执行落地

重点识别：

* 真正干活的人
* 能独立推进的人
* 能解决问题的人

避免：

* 只会讲策略
* PPT 型候选人
* 模糊表达型候选人

---

# 使用方式

输入：

```text
岗位 JD
+
候选人简历 / 背景信息
```

即可自动生成：

* 面试评估
* 深度问题
* 风险项
* 快速验证清单
